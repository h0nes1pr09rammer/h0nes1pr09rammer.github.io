jacoco手工测试覆盖率工具使用方法

Veröffentlicht am 2017-03-28 | in 工具使用

原文出处

环境：android studio+ gradle+jacoco

步骤：创建ec文件

在工程的oncreate()方法添加如下代码
String DEFAULT_COVERAGE_FILE_PATH = "/mnt/sdcard/coverage.ec";

   File file = new File(DEFAULT_COVERAGE_FILE_PATH);

   if (!file.exists()) {
       try {
           file.createNewFile();
       } catch (IOException e) {
           e.printStackTrace();
       }
   }

步骤：写入数据

在工程的ondestory()方法添加如下代码
OutputStream out = null;
try {
    out = new FileOutputStream("/mnt/sdcard/coverage.ec", false);
    Object agent = Class.forName("org.jacoco.agent.rt.RT")
            .getMethod("getAgent")
            .invoke(null);

    out.write((byte[]) agent.getClass().getMethod("getExecutionData", boolean.class)
            .invoke(agent, false));
} catch (Exception e) {
    Log.d(TAG, e.toString(), e);
} finally {
    if (out != null) {
        try {
            out.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

步骤：解析报告task编写

apply plugin: 'jacoco'

 release {
        testCoverageEnabled = true
  }
 //单项目工程
task jacocoTestReport (type:JacocoReport) {

    group = "Reporting"

    description = "Generate Jacoco coverage reports on the  build."



    classDirectories = fileTree(

            dir: "${project.buildDir}/intermediates/classes",
             //去除多余class文件
            excludes: ['**/R.class',

                       '**/R$*.class',

                       '**/*$ViewInjector*.*',

                       '**/BuildConfig.*',

                       '**/Manifest*.*']

    )



    def coverageSourceDirs = [

            "src/main/java"

    ]

    additionalSourceDirs = files(coverageSourceDirs)

    sourceDirectories = files(coverageSourceDirs)

    executionData = fileTree(dir:'./build/outputs',include:'**/*.ec')



    reports {

        xml.enabled = true

        html.enabled = true

    }

}
//多项目工程（读取aar中代码覆盖率）
task jacocoTestReport (type:JacocoReport) {

    group = "Reporting"

    description = "Generate Jacoco coverage reports on the  build."

    classDirectories = fileTree(
            //获取整个工程中class文件
            dir:  rootDir.getAbsolutePath(),
            include: '*/build/intermediates/classes/**/*.class',
            excludes: ['**/R.class',
                       '**/R$*.class',
                       '**/*$ViewInjector*.*',
                       '**/BuildConfig.*',
                       '**/Manifest*.*']
    )

       def coverageSourceDirs = [

                "src/main/java"

        ]

        additionalSourceDirs = files(coverageSourceDirs)

        sourceDirectories = files(coverageSourceDirs)

    executionData = fileTree(dir:'./build/outputs',include:'**/*.ec')

    reports {
        xml.enabled = true
        html.enabled = true
    }
}

步骤：将报告从手机里导入到D：\test

adb pull /sdcard/coverage.ec d:\test

步骤：.ec文件放到./build/outputs目录下

gradlew jacocoTestReport

注意事项：

  • gradlew命令运行时显示下载gradle，可将本地对应gradle版本复制到C:\Users\CTWLPC.gradle\wrapper\dists\对应版本下
  • build.gradle中buildtype目录下添加testCoverageEnabled = true
